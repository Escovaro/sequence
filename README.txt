Bedienungsanleitung:

1. Diesen Ordner in eigenen /htdocs/-Ordner ablegen
2. In MySQL neue Datenbank "sqdb" erstellen, ohne Passwort
3. Datenbankfile in diese Datenbank importieren
4. Fertig

Filefunktionen werden in den jeweiligen Unterordnern nochmals in Readme-Dateien erläutert.

FAQ: 
Falls Fehlermeldung Projekterstellung: Check, ob Auto-Inkremente in DB übernommen wurden, Projekttitel auf "UNIQUE" setzen.
DB nicht Case-Sensitive!
